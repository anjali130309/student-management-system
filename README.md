# Student Management System

A console-based Student Management System developed in C++ using Object-Oriented Programming and File Handling concepts.

---

## Features

- Add Student
- View Student Records
- Search Student by ID
- Update Student Details
- Delete Student Records
- Permanent Data Storage using File Handling

---

## Technologies Used

- C++
- STL (Vector)
- File Handling
- OOP Concepts

## How to Run

### Compile the Program

```bash
g++ main.cpp -o sms
```

### Run the Program

#### Windows

```bash
.\sms.exe
```

#### Linux / Mac

```bash
./sms
```
